PruneYourFollows is a web app built with Xata that lets you find accounts on Twitter to unfollow. It uses Xata's search capabilities to find accounts that you follow that are inactive or stale and lets you unfollow them in bulk.

![PruneYourFollows](images/pyf.png)

- [Explore the app](https://pruneyourfollows.com)
- [View the source code](https://github.com/queen-raae/prune-your-follows)

## Tech stack

- [Gatsby](https://www.gatsbyjs.com/) - React-based framework for building static websites and web applications.
- [TailwindCSS](https://tailwindcss.com) - Utility-first CSS framework for style.

### Features

- **[Prune your follows](/docs/examples/prune-your-follows):** Develop a web app using Xata, GatsbyJS, NextAuth, and Tailwind to find Twitter accounts to unfollow.
