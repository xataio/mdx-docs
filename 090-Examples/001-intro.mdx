## What can you build with Xata?

Xata is a versatile platform that can be used for various purposes, and we provide a range of resources to help you get started.
In addition to our [starter framework apps](/docs/getting-started/intro), you can craft custom solutions tailored to your requirements. Explore our example apps to gain insights into how Xata can help in your development projects.

We've gathered practical examples to show you how Xata works with different technologies and services.

### Sample Use Cases

Our samples provide step-by-step guidance for using Xata in your projects. Whether it's user authentication or handling image and file uploads and storage, these examples offer a clear solution.

- Gallery app: Build an image catalog with Xata, Next.js & Chakra UI
- XMDB: Create a movie database

### Starter Applications

Xata's starter applications serve as ready-made project foundations. These GitHub repositories are easy to clone or fork, and give you a solid starting point for your projects. Each starter includes a detailed README explaining how to set up the application.

Explore these starter options:

Next.js
Remix + Netlify
Vercel Serverless Functions

You can use a starter as-is or customize it to fit your project's unique needs.

For a comprehensive list of examples and starters, visit our GitHub examples repository.
