The Xata starter guides for [Astro](/docs/getting-started/astro), [Next.js](/docs/getting-started/nextjs), [Nuxt](/docs/getting-started/nuxt), [Remix](/docs/getting-started/remix), [SolidStart](docs/getting-started/solidstart), and [SvelteKit](docs/getting-started/sveltekit) walk you through building a blog application and help you learn some core Xata fundamentals. Additionally, the guides explore framework-specific features and how they work with Xata.

## What will you learn?

The guides demonstrate the following features:

- Using the Xata CLI to create a database
- Defining a database schema and seeding the database by importing a CSV
- Generating TypeScript code with database branching
- Querying and filtering
- Using full-text fuzzy search
